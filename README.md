### Overview
Real-time tool that builds and visualizes full **implied volatility (IV) surfaces** from live options chains. Instantly see volatility smile, skew, and term structure for any ticker.

### Key Features
- Interactive & rotatable 3D surface (Plotly)  
- Switch between **Strike** and **Moneyness** views  
- Full-color volatility heatmap  
- Works with any stock/ETF with options (SPY, AAPL, TSLA, PLTR, NVDA, etc.)  
- Live data via **yfinance** (free)  
- One-click export â†’ PNG / HTML / PDF
